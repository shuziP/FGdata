## 期货

### 期货新闻数据接口

#### 新浪财经-期货品种资讯

接口：futures_news_sina

目标地址：https://finance.sina.com.cn/futuremarket/
（注意：实际访问的URL会根据期货品种代码动态改变，如https://finance.sina.com.cn/money/future/HC/）

描述：
从新浪财经网站获取指定期货品种的最新资讯信息，包括新闻标题、发布日期和链接等。

输入参数：
futures_code（字符串）：期货品种的代码，如HC（焦炭）、CU（铜）等。根据此代码动态生成访问的URL。

输出参数：
DataFrame：返回一个Pandas DataFrame对象，包含以下列：
期货品种：期货的品种名称（有时可能与输入的futures_code相同，也可能根据网页内容有所不同）。
新闻标题：新闻资讯的标题。
发布日期：新闻资讯的发布日期，格式可能因网页内容而异，可能需要进一步处理。
链接：新闻资讯的链接地址。

接口示例： 
# 调用接口获取HC期货品种的资讯
```python
import fgdata  
futures_code = 'HC'
df = your_module_name.fetch_futures_news(futures_code)

print(df.head())
```
